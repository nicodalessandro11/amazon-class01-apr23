Lists

HTML lists allow web developers to group a set of related items in lists. At the end of the day our job is to properly and cretively display data within the document.

Commonly, we have two approaches of creating lists to render data

Ordered Lists

Unordered Lists

Syntax

Ordered Lists

  1. List item
  2. List item
  3. List item

Unordered Lists

  • List item
  • List item
  • List item

We can combined these types of lists in whatever way we like, but remember, good UX is always needed.

Example: Nested List

  • Item 1
  • Item 2
    • Sub Item 2.1
  • Item 3
  • Item 4
